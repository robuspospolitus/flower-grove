# Flover Grove

Flower Grove is a front-end project focused on building a clean, visually consistent user interface using modern React and SCSS.

The application serves as a portfolio piece demonstrating UI implementation, component structure, and styling architecture.

## About the Project

The project presents a thematic landing page centered around flowers, combining structured layout design with aesthetic presentation.

It was created to showcase:

- ability to translate design ideas into code
- consistent styling using SCSS
- reusable React component structure
- responsive layout implementation

### Tech Stack

- React - component-based architecture
- SCSS (Sass) - modular and maintainable styling
- JavaScript

## Getting started

### Website
The project is also available from [HERE](https://flower-grove.vercel.app/)

### Installation and setup

Clone the repository

```
git clone https://github.com/robuspospolitus/recruitment-frontend-solvro.git
```

Go into the directory, install dependencies and run the app
```
cd flower-grove
npm install
npm start
```
